# taskcli

A simple to-do list cli app

## how to install taskcli
```
git clone https://github.com/Robin3880/taskcli
cd taskcli
pip install .
```
## taskcli commands
taskcli add "task"

taskcli delete "task"

taskcli mark-in-progress "task"

taskcli mark-done "task"

taskcli list

taskcli list done

taskcli list in-progress

taskcli list to-do
